How to test :

  • mouseLeft + mouseMove : Move the spherical camera.
  • mouseWheel : Zoom.
  • L : Toggle all the lights.
  • A : Toggle the blue directionnal light.
  • B : Toggle the yellow directionnal light.
  • G : Toggle the grid.
